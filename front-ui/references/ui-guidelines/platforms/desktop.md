# Platforms — Desktop

## When to consult this file

- Viewports ≥ 1024 px
- Keyboard + mouse / trackpad primary inputs

## Defaults

- **Multi-column** layouts encouraged; respect `max-w-6xl` for reading-heavy content.
- **Sidebar nav** preferred over top nav for app-style products with many destinations.
- **Keyboard shortcuts surfaced**: command palette (`⌘K`), inline shortcut hints next to menu items.
- **Hover state polish**: subtle background tints and tooltips for icon-only buttons.
- **Right-click menus** wherever the user might expect them (rows, items, files).

## Patterns

- 3-column layout: sidebar / list / detail.
- Floating action button is rare on desktop — use a clear nav-bar action instead.
- Skeletons must match the multi-column shape, not the mobile-stacked shape.
