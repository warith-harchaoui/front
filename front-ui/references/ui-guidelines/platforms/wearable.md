# Platforms — Wearable

## When to consult this file

- Small-screen targets (watch faces, fitness bands)
- Glanceable surfaces

## Defaults

- **Glanceable above interactive.** Most of the time the user just looks.
- **High contrast, big type** — body ≥ 18 px.
- **One primary action per screen.** Secondary actions hide in a swipe.
- **No long forms.** Wearable is for confirmations, not data entry.

## Patterns

- Vertical list of large rows (`min-h-14`, `text-[19px]`).
- Single full-screen action ("Pay", "Start workout").
- Notifications expand to a single-action sheet.

## Checklist

- [ ] Body ≥ 18 px.
- [ ] One primary action per screen.
- [ ] No long form input.
- [ ] Reads at arm's length glance (≥ 3:1 contrast everywhere).
