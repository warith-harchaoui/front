# Requirements for install_captions.py and captions_from_whisper.py
#
# Install:
#   pip install -r scripts/requirements-captions.txt
#
# Used for:
#   - vocal-helper : the project author's whisper.cpp over-layer. Owns the
#                    model defaults, word-timestamp wiring, hallucination
#                    guard and the vocabulary-biasing prompt lever. Pulls
#                    ``pywhispercpp`` (whisper.cpp binding, pre-compiled
#                    wheels) transitively — captions no longer touch
#                    pywhispercpp directly.
#   - audio-helper : 16 kHz mono decode (``load_audio``) + WAV conversion
#                    (``sound_converter``) for audio sources.
#   - video-helper : audio-track extraction (``extract_audio_track``) for
#                    video sources.
#   - click        : CLI parser for captions_from_whisper.py; install_captions
#                    keeps its bespoke flow.
#   - langdetect   : detect language from vocabulary source text so the
#                    composed initial_prompt opener matches.
#
# The three ``*-helper`` packages are on PyPI, pinned with a lower bound so
# their public API (function names, signatures) stays stable while still
# picking up fixes. ``vocal-helper`` pins ``audio-helper`` / ``os-helper``
# from PyPI itself, so these direct references dedupe cleanly against its
# own — no ``.git@vX`` URL form needed any more. ``vocal-helper``'s base
# install already pulls ``pywhispercpp`` (STT) + ``silero-vad``, so no
# extra is required for the caption path.

click>=8.1
langdetect>=1.0.9
vocal-helper>=0.6.0
audio-helper>=1.6.0
video-helper>=1.7.0
