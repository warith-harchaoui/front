# CLI → web GUI — `front-cli-gui`

Wrap an existing command-line tool in a single-page vanilla-JS + Tailwind GUI. It reads the CLI's argparse / Click / clap / cobra parser and maps sub-commands and flags to forms, segmented controls, file inputs, and streaming log panels.

This is the human landing page. It points to the three places that hold the
detail — nothing is duplicated here.

- **What it is & what activates it** — [`front-cli-gui/SKILL.md`](../front-cli-gui/SKILL.md)
  (the agent-facing spec: purpose, trigger phrases, full flag surface).
- **Run it** — [`EXAMPLES.md`](../EXAMPLES.md) has a copy-paste recipe for
  `front-cli-gui`.
- **Go deeper** — [`front-cli-gui/references/`](../front-cli-gui/references/): the CLI→GUI workflow, host wiring, hardening.
