WCAG contrast audit with OKLCH-neighbour fixes, color-vision-deficiency simulation, a curated Apple-inspired palette with semantic projections, perceptual lighten / darken, and a Tailwind theme emitter. Deterministic — no model, no network.

detail — nothing is duplicated here.
