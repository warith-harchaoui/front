# UI Guidelines — Reference Index

Curated, platform-native UI guidance for vanilla JavaScript + Tailwind CSS output. Each file distills one topic into actionable rules, with concrete Tailwind / JS patterns.

## How Claude should use this index

1. Identify which topics apply to the task using the map below.
2. Read **only** the files that apply. Do not preload everything.
3. SKILL.md should reference these files by relative path; do not duplicate their content.
